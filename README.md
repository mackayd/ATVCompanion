
# ATVCompanion (repo skeleton)

Windows companion app to wake and control Android/Google TVs with a **plugin-based control layer**.
- **Wake**: Wake-on-LAN (WoL/WoWLAN)
- **Control plugins** (modular): Philips (JointSPACE v6), Sony (BRAVIA JSON-RPC/IRCC-IP), etc.
- **Components**: Core (SDK), Windows Service, WPF UI, CLI, (optional) Installer.

> This is a **buildable skeleton** with stubs and an initial Philips implementation. Flesh out features incrementally.
